# Image-Detector-Swift-coreML

## Introduction: 
This repository contains an iOS app that can detect objects from images up to a reasonably accurate level.

## Tech Stack Used:
1. Swift
2. coreML
3. MobileNetV2

## Tested On:
iPhone 14 Plus with multiple flower and leaf images.
